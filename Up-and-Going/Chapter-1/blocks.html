Blocks

A grouped series of statements is called a block. In JS, a block is defined by wrapping one or more statments inside a curly-brace pair {..}.

Consider:

var amount = 99.99;

// a general block

{
amount = amount *2;
console.log( amount ); // 199.98
}

Typically, blocks are attached to some other control statement, such as an if statement or a loop. For example:

var = 99.99;

// is amount big enough?

if (amount > 10) { // block attached to 'if'
amount = amount * 2;
console.log ( amount ); // 199.98
}

The statements inside the block will only be processed if the conditional passes.
