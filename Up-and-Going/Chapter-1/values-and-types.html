Values and Types

Different representations for values are called types in programming terminology. JS has built-in types for each of these so called primitive values

  • For Math: number
  • To print a value on the screen: string (one or more characters, words, sentences)
  • To make a decision in your program: boolean (true or false)

Converting Between Types

If you have a number but need to print it on the screen, you need to convert the value to a string. This is called "coercion".
